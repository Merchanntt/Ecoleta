## Getting_Started
To use this app, you'll need...

- Clone this repository: 
```shell
cd "dir from your preference"
git clone https://github.com/Merchanntt/Next-Level-Week-01
``` 
- Run: 
```shell
Yarn
```
- Create the dataBase:
```shell
cd.. backend
yarn knex:migrate
yarn knex:seed
```
- And finally, run the servers:
```shell
cd mobile
yarn start
-
cd backend
yarn dev:server
-
cd frontend
yarn start
```

Feel free to change on your one way <3
